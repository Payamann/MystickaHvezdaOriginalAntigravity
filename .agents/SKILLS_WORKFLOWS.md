# Common Skill Workflows

**Version:** 1.0.0
**Purpose:** Document recommended sequences of skills for common use cases

These workflows show which skills to use together and in what order for maximum effectiveness.

---

## Workflow 1: SEO Site Launch (Complete Site)

**Goal:** Launch a new website optimized for SEO from day one
**Duration:** 2-3 weeks (depending on site size)
**Complexity:** High
**Best For:** New websites, complete redesigns

### Phase 1: Planning & Research (Days 1-3)

1. **keyword-research** (3-4 hours)
   - Identify target keywords by topic
   - Assess search volume and difficulty
   - Create keyword clusters
   - Output: Keyword opportunities spreadsheet

2. **competitor-analysis** (2-3 hours)
   - Analyze top 5 competitors
   - Identify content gaps
   - Document competitor backlink strategy
   - Output: Competitive analysis report

3. **seo-plan** (2-3 hours)
   - Create 6-12 month SEO roadmap
   - Define phases and milestones
   - Allocate resources
   - Output: SEO roadmap document

4. **content-strategy** (2-3 hours)
   - Plan content pillars and clusters
   - Create editorial calendar
   - Define success metrics
   - Output: Content calendar and strategy

### Phase 2: Architecture & Structure (Days 4-5)

5. **site-architecture** (3-4 hours)
   - Design URL structure
   - Plan information hierarchy
   - Create navigation strategy
   - Define breadcrumb structure
   - Output: Site architecture diagram and documentation

6. **seo-sitemap** (1-2 hours)
   - Generate XML sitemap template
   - Ensure all pages are included
   - Output: Sitemap.xml

### Phase 3: Content Creation (Days 6-15)

7. **seo-content-writer** (2-3 hours per page)
   - Write homepage copy
   - Write pillar pages
   - Write supporting cluster content
   - Output: Finished, optimized content

8. **copywriting** (1-2 hours per page if needed)
   - Optimize messaging
   - Improve CTAs
   - Test value propositions
   - Output: Polished copy

### Phase 4: Technical Optimization (Days 16-17)

9. **schema-markup-generator** (1-2 hours)
   - Add structured data to pages
   - Implement schema by page type
   - Validate schema markup
   - Output: JSON-LD markup code

10. **technical-seo-checker** (2-3 hours)
    - Check crawlability
    - Verify indexability
    - Check Core Web Vitals
    - Implement fixes
    - Output: Technical audit report and fixes

### Phase 5: Validation (Days 18-21)

11. **seo-audit** (4-6 hours)
    - Full comprehensive audit
    - Identify any remaining issues
    - Create final checklist
    - Output: Pre-launch SEO audit report

12. **alert-manager** (1 hour)
    - Set up ranking monitoring
    - Configure traffic alerts
    - Define response plans
    - Output: Alert configuration

### Success Metrics
- All pages indexed within 2 weeks
- 80%+ crawl efficiency
- 0 crawl errors
- 90+ on SEO score
- Core Web Vitals in green
- All pages have proper schema

## Workflow 2: SEO Content Refresh Campaign

**Goal:** Update and optimize existing content for better rankings
**Duration:** 1-2 weeks (depends on content volume)
**Complexity:** Medium
**Best For:** Established websites with underperforming content

### Phase 1: Analysis (Days 1-2)

1. **seo-audit** (3-4 hours)
   - Identify underperforming pages
   - Find content gaps vs competitors
   - Identify technical issues
   - Output: Pages to refresh priority list

2. **serp-analysis** (2-3 hours)
   - Analyze current top 10 results
   - Identify content gaps
   - Understand ranking factors
   - Output: SERP competitive analysis

3. **competitor-analysis** (2-3 hours)
   - See what competitors have improved
   - Find new competitor topics
   - Identify messaging changes
   - Output: Competitive changes report

### Phase 2: Optimization (Days 3-8)

4. **seo-content-writer** (2-3 hours per page)
   - Rewrite underperforming pages
   - Add missing sections
   - Improve depth and coverage
   - Output: Refreshed page content

5. **copywriting** (1-2 hours per page if needed)
   - Improve headlines
   - Enhance CTAs
   - Better value communication
   - Output: Optimized copy

6. **content-refresher** (1-2 hours per page)
   - Update outdated information
   - Add recent statistics
   - Update examples
   - Output: Updated page

### Phase 3: Technical Updates (Days 9-10)

7. **meta-tags-optimizer** (30 min per page)
   - Update title tags
   - Refresh meta descriptions
   - Improve CTR potential
   - Output: New meta tags list

8. **schema-markup-generator** (30 min per page)
   - Update schema markup
   - Add new relevant schema types
   - Output: Updated schema code

### Phase 4: Validation (Days 11-14)

9. **seo-page** (30 min per page)
   - Validate each refreshed page
   - Check for issues
   - Verify improvements
   - Output: Page validation report

10. **alert-manager** (optional - if new pages)
    - Update monitoring for changed pages
    - Set new tracking baselines
    - Output: Updated alerts

### Success Metrics
- 20%+ improvement in target keywords within 4-8 weeks
- Increased click-through rates (higher CTR in search)
- Reduced bounce rate
- Longer average session duration
- Content freshness signals updated

## Workflow 3: Programmatic SEO Implementation

**Goal:** Build and optimize pages at scale using templates and data
**Duration:** 2-4 weeks (depends on page count)
**Complexity:** High (technical)
**Best For:** Generating 50-10,000+ similar pages

### Phase 1: Planning (Days 1-3)

1. **keyword-research** (4-6 hours)
   - Identify keyword patterns
   - Assess search volume across patterns
   - Create keyword clusters
   - Output: Pattern spreadsheet with volume

2. **programmatic-seo** (4-6 hours)
   - Choose playbook (templates, locations, comparisons, etc.)
   - Define data sources
   - Plan URL structure
   - Create quality gates
   - Output: pSEO strategy document

3. **site-architecture** (2-3 hours)
   - Plan URL hierarchy
   - Define breadcrumbs
   - Plan internal linking
   - Output: Architecture documentation

### Phase 2: Content & Data (Days 4-10)

4. **seo-content-writer** (2-4 hours)
   - Create content templates
   - Define unique content sections
   - Create conditional content rules
   - Output: Template structure and samples

5. **schema-markup-generator** (1-2 hours)
   - Create schema templates
   - Define dynamic schema fields
   - Output: Schema JSON-LD templates

6. Data preparation (developer task)
   - Compile/organize data
   - Validate data quality
   - Create data feeds
   - Output: Database or CSV with data

### Phase 3: Technical Implementation (Days 11-16)

7. Implementation (developer task)
   - Build page generation system
   - Implement templates
   - Generate pages from data
   - Create sitemap

### Phase 4: Optimization & Launch (Days 17-21)

8. **technical-seo-checker** (3-4 hours)
   - Verify indexability of generated pages
   - Check for crawl issues
   - Validate technical implementation
   - Output: Technical validation report

9. **seo-audit** (4-6 hours)
   - Audit sample of generated pages
   - Check for thin content
   - Verify quality gates
   - Output: Audit report and fixes

10. **alert-manager** (1-2 hours)
    - Set up monitoring for programmatic pages
    - Track indexation rate
    - Monitor ranking performance
    - Output: Alert configuration

### Success Metrics
- 90%+ pages indexed within 2-4 weeks
- Average ranking position improving over 2-3 months
- No manual action penalties
- Consistent quality across generated pages
- Traffic growth from new pages within 3-6 months

## Workflow 4: Homepage & Landing Page Optimization

**Goal:** Improve conversion rate on key pages
**Duration:** 1-2 weeks
**Complexity:** Medium
**Best For:** Improving conversion performance

### Phase 1: Analysis (Days 1-2)

1. **seo-page** (2-3 hours)
   - Deep dive on current page
   - Identify SEO gaps
   - Check rankings for page
   - Output: Page analysis report

2. **serp-analysis** (1-2 hours)
   - Check competitor pages
   - Identify messaging differences
   - Find gaps in your positioning
   - Output: SERP competitive analysis

### Phase 2: Copy & UX (Days 3-7)

3. **copywriting** (3-4 hours)
   - Rewrite headlines
   - Improve messaging
   - Strengthen CTAs
   - Add social proof
   - Output: Revised copy

4. **page-cro** (3-4 hours)
   - Review page structure
   - Optimize element hierarchy
   - Improve flow
   - Test variations
   - Output: CRO recommendations

5. **marketing-psychology** (1-2 hours optional)
   - Apply psychological principles
   - Test persuasion elements
   - Optimize framing
   - Output: Psychology-based recommendations

### Phase 3: Implementation (Days 8-10)

6. Implementation (design/dev task)
   - Update page copy
   - Implement design changes
   - Deploy variations for testing

### Phase 4: Testing & Monitoring (Days 11+)

7. **ab-test-setup** (2-3 hours)
   - Design A/B test
   - Set up variants
   - Define success metrics
   - Output: Test plan

8. **analytics-tracking** (1-2 hours)
   - Ensure proper event tracking
   - Set up conversion tracking
   - Define KPIs
   - Output: Tracking implementation

### Success Metrics
- 10-30% improvement in conversion rate within 2-4 weeks
- Lower bounce rate
- Increased time on page
- Higher average order value (if applicable)
- Better scroll depth

## Workflow 5: Competitive Advantage Research

**Goal:** Find market gaps and competitive opportunities
**Duration:** 3-5 days
**Complexity:** Medium
**Best For:** Market analysis, content strategy, product planning

### Phase 1: Competitive Landscape (Days 1-2)

1. **competitor-analysis** (3-4 hours)
   - Identify direct competitors
   - Analyze their strategies
   - Identify gaps and weaknesses
   - Output: Competitor analysis matrix

2. **serp-analysis** (2-3 hours)
   - Analyze keyword rankings
   - See what competitors rank for
   - Identify ranking opportunities
   - Output: Ranking opportunities list

3. **backlink-analyzer** (2-3 hours)
   - Analyze competitor backlinks
   - Find link opportunities
   - Identify authority sources
   - Output: Link opportunity report

### Phase 2: Content & Keyword Gaps (Days 2-3)

4. **content-gap-analysis** (3-4 hours)
   - Identify topics you're missing
   - See what competitors cover
   - Find expansion opportunities
   - Output: Content gap report

5. **keyword-research** (3-4 hours)
   - Find keyword opportunities
   - Identify low-competition keywords
   - Cluster related keywords
   - Output: Opportunity keyword list

### Phase 3: Strategy (Days 4-5)

6. **competitor-alternatives** (2-3 hours)
   - Research alternative solutions
   - Find underserved niches
   - Identify unique positioning
   - Output: Alternatives analysis

7. **content-strategy** (2-3 hours)
   - Plan content to fill gaps
   - Create topics list
   - Prioritize by opportunity
   - Output: Content roadmap

### Success Metrics
- Identified 10+ content opportunities vs competitors
- Found 5+ underserved niches
- Clear differentiation strategy
- Priority content roadmap for next 3-6 months

## Workflow 6: International/Multi-Language SEO

**Goal:** Optimize website for multiple languages/regions
**Duration:** 2-3 weeks
**Complexity:** High (technical)
**Best For:** Global websites, multi-region presence

### Phase 1: Planning (Days 1-2)

1. **seo-plan** (2-3 hours)
   - Define target markets/languages
   - Assess localization needs
   - Plan resource allocation
   - Output: International SEO roadmap

2. **keyword-research** (4-6 hours)
   - Research keywords by language
   - Identify regional variations
   - Assess search behavior differences
   - Output: Multi-language keyword matrix

### Phase 2: Content (Days 3-10)

3. **seo-content-writer** (varies by volume)
   - Create content for each language/region
   - Localize (not just translate)
   - Adapt to local preferences
   - Output: Multi-language content

4. **geo-content-optimizer** (2-3 hours per language)
   - Optimize for regional keywords
   - Add local relevance signals
   - Local social proof and trust signals
   - Output: Localized content

### Phase 3: Technical Implementation (Days 11-15)

5. **seo-hreflang** (3-4 hours)
   - Implement hreflang tags
   - Set up alternate language links
   - Validate implementation
   - Output: Hreflang implementation guide

6. **site-architecture** (2-3 hours)
   - Plan URL structure (subdomain, subfolder, TLD)
   - Create navigation for language selection
   - Plan breadcrumbs
   - Output: Architecture documentation

### Phase 4: Validation (Days 16-21)

7. **technical-seo-checker** (3-4 hours)
   - Verify hreflang setup
   - Check language detection
   - Validate technical implementation
   - Output: Technical report

8. **seo-audit** (4-6 hours)
   - Full multi-language audit
   - Identify regional issues
   - Output: Multi-language audit report

### Success Metrics
- Proper language version appearing in SERPs
- No indexation issues across versions
- Regional traffic from target countries within 4-8 weeks
- Hreflang validation: 0 errors

## Workflow Best Practices

### 1. Timeline Management
- Don't skip phases (sequential dependencies matter)
- Budget time for iteration and fixes
- Build in buffer for unexpected issues

### 2. Team Coordination
- Assign clear ownership for each phase
- Define handoff points
- Maintain clear communication

### 3. Quality Checkpoints
- Validate outputs before moving to next phase
- Use success metrics to track progress
- Iterate if metrics not met

### 4. Documentation
- Keep detailed notes of decisions
- Document assumptions made
- Record actual timelines vs estimates

### 5. Monitoring
- Set up alerts from the start
- Track metrics continuously
- Be ready to pivot based on data

## Extending Workflows

These workflows can be combined or adapted:

- **Site launch + programmatic SEO** = Launch with pages at scale
- **Content refresh + CRO** = Improve rankings AND conversions
- **Research + strategy** = 2-week market analysis deep-dive
- **International + refresh** = Localize existing site

Mix and match skills based on your specific needs!
